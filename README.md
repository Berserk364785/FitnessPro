FitPulse

AI-тренер для домашних тренировок

FitPulse — это веб-приложение для тренировок с использованием компьютерного зрения и искусственного интеллекта.

Приложение анализирует движения через камеру устройства, автоматически считает повторения, помогает следить за техникой выполнения упражнений и сохраняет прогресс тренировок.

✨ Возможности

- 📈 Отслеживание прогресса тренировок

📸 Как это работает

1. Открой приложение.
2. Разреши доступ к камере.
3. Выбери упражнение.
4. Выполняй движения перед камерой.
5. Получай анализ техники и статистику в реальном времени.
